Safe, self-hosted fake-malicious pages for live demos. You never visit these directly — copy a URL and detonate it in Sandboxed. Each one reliably triggers specific signals.

Before you present: open the app, run one warm-up detonation (the first one is slow — cold server + first Chromium launch). For the password-trap reveal, turn on Password trap in Options; for the video, leave Record replay on. Recommended: also enable Phishing detector. These pages are noindex and marked "SANDBOX DECOY", so they never affect your real site.
